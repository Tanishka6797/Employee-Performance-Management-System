# Employee-Performance-Management-System

A] Project Description :
The Employee Management System is a DBMS mini project developed using Python and SQL to manage employee records efficiently. The system supports core CRUD operations through a simple and user-friendly GUI, ensuring structured storage and easy retrieval of employee data.

B] Features :
1.Add new employee records
2.View existing employee details
3.Update employee information
4.Delete employee records
5.GUI-based interaction
6.Structured and secure database storage

C] Tech Stack :
1.Python
2.SQL / MySQL
3.DBMS Concepts
4.Python GUI
5.VS Code

D] Database Concepts Used :
1.Relational database design
2.Primary and foreign keys
3.Normalization
4.SQL queries (INSERT, SELECT, UPDATE, DELETE)

E] Team Contribution :
This project was developed collaboratively. Responsibilities were shared across database design, backend logic, and GUI integration, helping us gain practical experience in teamwork and real-world application development.
